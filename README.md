You want to send attack plans to your fellow commanding officer in the Imperial Army, but- Oh no! A rebel spy has intercepted your plans!
Fear not, for this program will leave them more confused than before
Because your plans are not words, but a single image

MOM (Mine, Only Mine) uses LSB Steganography in order to encrypt messages (attack plans, passwords, state secrets) within the bytes of an image, hidden from the human eye.
Accompanying this software is the decoder, which lets you read the message sent by your comrade.
